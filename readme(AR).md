# 🚀 قالب كورسز زون - موقع أكاديمية متكامل

## 📝 وصف القالب
قالب موقع متكامل للأكاديميات ومراكز التدريب، يشمل:
- نظام إدارة كورسات (إضافة، تعديل، حذف)
- نظام إدارة مدرسين (إضافة، تعديل، حذف)
- نظام إدارة آراء الطلاب
- نظام إدارة فروع الأكاديمية
- مدونة (Blog) كاملة مع نظام إدارة مقالات
- لوحة تحكم (Admin Panel) لإدارة كل المحتوى
- دعم كامل للغة العربية والإنجليزية
- تصميم متجاوب مع جميع الأجهزة

## 🎨 تغيير الألوان
افتح ملف `css/style.css` وغير المتغيرات في قسم `:root`

مثال:
```css
:root {
    --primary: #8E11B2;      /* اللون الأساسي */
    --primary-light: #D6B7D5; /* اللون الفاتح */
    --text-dark: #2A1B2F;     /* لون النصوص */
}
