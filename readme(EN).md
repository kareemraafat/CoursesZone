# 🚀 Courses Zone Template - Complete Academy Website

## 📝 Description
Complete website template for academies and training centers, including:
- Course Management System (Add, Edit, Delete)
- Instructors Management System
- Student Testimonials System
- Branches Management System
- Full Blog System with Article Management
- Admin Panel for all content management
- Full Arabic/English Support
- Responsive Design for all devices

## 🎨 Changing Colors
Open `css/style.css` and modify the variables in the `:root` section

Example:
```css
:root {
    --primary: #8E11B2;      /* Primary color */
    --primary-light: #D6B7D5; /* Light color */
    --text-dark: #2A1B2F;     /* Text color */
}


🖼️ Changing Logo
Delete images/logo-placeholder.png

Place your logo image with the same name

Or change the path in index.html

📝 Changing Basic Text
Open js/config.js and modify the following values:

const SITE_CONFIG = {
    siteName: 'Your Site Name',
    siteNameAr: 'Your Site Name in Arabic',
    whatsappNumber: 'Your WhatsApp Number',
    phoneNumber: 'Your Phone Number',
    email: 'Your Email'
};




📚 Adding New Courses
Go to Admin Panel: https://yoursite.com/admin.html

Fill in the data (Title, Price, Description, Image)

Click "Add Course"



✍️ Adding New Blog Posts
Go to Blog Admin Panel: https://yoursite.com/admin-blog.html

Write the article title and content

Add cover image and video link (optional)

Click "Add Post"



Deploying the Website
Create an account on GitHub

Create an account on Cloudflare

Upload files to GitHub

Connect GitHub to Cloudflare Pages

Your site will be live at: your-site-name.pages.dev
